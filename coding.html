Incorporating a Typewriter Effect with Dynamic Color Change in Hero Text using JavaScript

                        
                const messageArray = ["My Name is Name Surname. <br> I am a Web Developer."];
                let textPosition = 0;
                const speed = 150;
                
                const message = () => {
                  document.querySelector("#text").innerHTML = messageArray[0].substring(0, textPosition);
                
                  if (textPosition++ != messageArray[0].length) {
                    setTimeout(message, speed); 
                  } else {
                    // stop color change when typing effect is finished
                    clearInterval(colorInterval);
                  }
                }
                
                window.addEventListener("load", message);
                
                const colorChange = () => {   
                  const text = document.querySelector("#text");
                  const randomColor = Math.floor(Math.random()*16777215).toString(16);
                  text.style.color = "#" + randomColor;
                }
                
                // call colorChange every 200ms
                const colorInterval = setInterval(colorChange, 200);

This JavaScript code snippet elegantly demonstrates a typewriter effect incorporated into the hero text of a webpage, further enhanced by a dynamic color change feature. It's an intriguing approach to making the hero text more engaging and attention-grabbing for website visitors. The code begins by defining several crucial variables: messageArray to store the message, textPosition to keep track of the current position in the message, speed to set the pace of typing, and colorInterval for scheduling the color change function. Following this setup, two central functions are introduced: message and colorChange. The message function is responsible for the typewriter effect. It progressively outputs the message one character at a time, giving the illusion of text being typed out live. This function utilizes JavaScript's setTimeout to delay each character's appearance, resulting in a natural, typewriter-like rhythm. Moreover, an if statement within the message function checks if the typing effect is complete; when finished, it terminates the interval that triggers the colorChange function. In contrast, the colorChange function, which is invoked every 200 milliseconds via setInterval, manipulates the text color. By generating a random color value in hexadecimal format, it ensures that the hero text color changes dynamically and unpredictably as the text is being 'typed' out. This adds an extra layer of visual appeal and dynamism to the typewriter effect. This code provides an interactive, engaging experience for website visitors, demonstrating the creative possibilities of JavaScript when used in web design and development.

In the creative process of web design, certain aesthetic elements can significantly enhance the visual appeal and user experience. This particular HTML code snippet, featuring SVG (Scalable Vector Graphics), serves as a great example. It was designed specifically for the hero section of a charity project, in an attempt to replicate a distinct curved effect found on the official WoodGreen charity website. This compelling visual cue aims to capture visitors' attention and encourage their interaction. The unique curve effect is the product of carefully crafted SVG code. Leveraging the flexibility of SVGs, the curve is intricately shaped to mimic the application of a border-radius, but exclusively to the bottom of the hero image. To achieve this sophisticated visual touch, I utilized an SVG generator tool, which allows for the creation of such custom shapes with ease. What's particularly noteworthy about this approach is its versatility. The curve, while initially designed for a specific project, could easily be adapted for a myriad of different designs and applications, providing a visually engaging transition between the hero section and the rest of the content. This is just one demonstration of the power of SVGs in web design, and their potential to create captivating, tailor-made visuals.
